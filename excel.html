Excel

A função principal do Excel é a de criar fórmulas e inserir cálculos, além de ser possível organizar os dados e configurar diversos tipos de gráficos.

10 Atalhos Excel
Ctrl A Abre uma pasta de trabalho
Ctrl L Localiza e substitui os dados de uma determinada célula
Alt H Vai para a página incial
Ctrl Z Desfaz ação recente
Alt C R Escolha cor de preenchimento
Ctrl X Corta a seleção
Alt N Abre a guia inserir
Ctrl Alt N Negrito
Alt A Vai para a guia dados
Alt W Vai para a guia exibir
